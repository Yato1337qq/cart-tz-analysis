# Задание 1: Анализ ТЗ корзины

## Найденные проблемы

### 1. Пункты 2 и 9
Противоречие: нельзя меньше 1, но можно 0  
Решение: уменьшение до 0 = удаление товара

### 2. Пункты 7 и 13
Цена фиксируется и обновляется одновременно  
Решение: цена всегда актуальная из каталога

### 3. Лимиты (1, 3, 4)
Логическая несогласованность ограничений  
Решение: общий лимит корзины имеет приоритет

### 4. Ошибка (п.6)
Неясно, какой лимит превышен  
Решение: уточнить сообщение

---

## Исправленная версия

- Можно уменьшать количество до 0 (удаление)
- Цена не фиксируется, а обновляется
- Есть общий лимит корзины
